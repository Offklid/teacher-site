# Сайт учителя

Личный сайт учителя: материалы для работы, удобный доступ для учеников и родителей.

## Структура проекта

- `index.html` — главная страница
- `css/style.css` — стили
- `js/main.js` — скрипты
- `README.md` — этот файл

## Как открыть сайт локально

1. Открой папку `teacher-site` в Cursor: **File → Open Folder** и выбери эту папку.
2. Чтобы посмотреть сайт в браузере: открой файл `index.html` двойным щелчком или через расширение **Live Server** в Cursor (правый клик по `index.html` → Open with Live Server).
