# 🧬 Сайт учителя химии и биологии

Современный интерактивный сайт для преподавания химии и биологии в 8-11 классах. Разработан с использованием современных веб-технологий и дизайна в стиле US startup.

## ✨ Основные возможности

### 📚 Учебные материалы
- **Автоматическая загрузка материалов** из Google Drive
- Фильтрация по классам (8-11)
- Презентации, конспекты, тесты и задания
- Удобный просмотр и скачивание файлов

### ⚗️ Интерактивная химия
- **Периодическая таблица Менделеева** (все 118 элементов)
  - Короткая форма (8 групп)
  - Интерактивные карточки элементов
  - Интересные факты о каждом элементе
  - Цветовая кодировка по типам элементов
- Подготовка к ОГЭ/ЕГЭ
- 16 интерактивных кроссвордов по химии

### 🔬 Интерактивная биология
- **3D модель человеческого тела**
- Интерактивное изучение анатомии
- Детальная информация об органах и системах
- Подготовка к ОГЭ/ЕГЭ

### 🎯 Дополнительные функции
- **Умный поиск** по всему сайту (Fuse.js)
- Интерактивные кроссворды с проверкой ответов
- Расписание и домашние задания
- Система отзывов (Cusdis)
- Адаптивный дизайн для всех устройств

## 🎨 Дизайн

- **Темная тема** с градиентами
- **Glassmorphism** эффекты
- **Анимации** (AOS, Particles.js)
- **Современная типографика** (Inter, Space Grotesk)
- **Плавные переходы** и hover-эффекты

## 🛠️ Технологии

### Frontend
- HTML5, CSS3, JavaScript (ES6+)
- Particles.js - анимированный фон
- AOS (Animate On Scroll) - анимации при скролле
- Fuse.js - нечеткий поиск
- Typed.js - анимация печати текста

### Интеграции
- Google Drive API - загрузка учебных материалов
- Cusdis - система комментариев
- Human Anatomy Atlas - 3D модель тела

## 🚀 Быстрый старт

### Локальный запуск

1. Клонируйте репозиторий:
```bash
git clone https://github.com/Offklid/teacher-site.git
cd teacher-site
```

2. Откройте `index.html` в браузере или используйте Live Server:
```bash
# Если у вас установлен Python
python -m http.server 8000

# Или используйте расширение Live Server в VS Code
```

3. Откройте в браузере: `http://localhost:8000`

### Настройка Google Drive

1. Создайте файл `js/config.js` на основе `js/config.example.js`
2. Добавьте свой API ключ Google Drive
3. Укажите ID папок с материалами для каждого класса

```javascript
const CONFIG = {
  GOOGLE_API_KEY: 'ВАШ_API_КЛЮЧ',
  DRIVE_FOLDERS: {
    chemistry: {
      8: 'ID_ПАПКИ_8_КЛАСС',
      9: 'ID_ПАПКИ_9_КЛАСС',
      10: 'ID_ПАПКИ_10_КЛАСС',
      11: 'ID_ПАПКИ_11_КЛАСС'
    },
    biology: {
      // аналогично
    }
  }
};
```

## 📊 Особенности реализации

### Периодическая таблица Менделеева
- Короткая форма (8 групп) - классический вид
- Периоды 4-7 разбиты на 2 ряда
- Элементы группы VIII (Fe-Co-Ni, Ru-Rh-Pd и т.д.) отображаются вместе
- Лантаноиды и актиноиды вынесены отдельно
- 118 элементов с полной информацией

### Интерактивные кроссворды
- 16 тематических кроссвордов по химии
- Автоматическая проверка ответов
- Подсказки и интересные факты
- Сохранение прогресса

### Умный поиск
- Поиск по всему контенту сайта
- Нечеткое совпадение (опечатки)
- Поиск по материалам, кроссвордам, страницам
- Мгновенные результаты
